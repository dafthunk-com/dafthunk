description: Generate and apply a new database migration
---

Help me create a new database migration:

1. Ask me what the migration is for
2. Generate the migration using `pnpm --filter '@dafthunk/api' db:generate`
3. Show me the generated SQL file
4. Apply it locally using `pnpm --filter '@dafthunk/api' db:migrate`
5. Update the schema types if needed
